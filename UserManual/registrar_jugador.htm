2.1. Registrar Jugador

Una vez ingresado desde el menú principal a la opción registrar jugador, se mostrará el siguiente formulario de registro con los campos que debe diligenciar. Al finalizar el diligenciamiento, active el botón para completar el Registro.

Select Género

Debe escoger entre la opción de Masculino o Femenino
 
Arriba

Input Apellido

Debe ingresar el apellido del jugador. Un máximo de 100 carácteres.
 
Arriba

Input Nombre

Debe ingresar el nombre del jugador. Un máximo de 100 carácteres.
 
Arriba

Select Posición

Debe escoger entre las opciones de posición para el jugador.
 
Arriba

Input Dirección

Debe ingresar la dirección del jugador.
 
Arriba

Input Teléfono

Debe ingresar un número telefónico válido de exactamente 10 caracteres.
 
Arriba

Select Fecha de nacimiento

Debe seleccionar la fecha de nacimiento del jugador a través del calendario.
 
 
Arriba

Input Documento de identidad

Debe introducir el documento de identidad de jugador.
 
Arriba

Input correo electrónico

Debe ingresar un correo electrónico válido.
 
Arriba

Input EPS

Debe ingresar la EPS del jugador
 
Arriba

Input Peso

Debe ingresar el valor del peso en números.
 
Arriba

Input Estatura

Debe ingresar el valor de la estatura en números.
 
Arriba

Botón Registrar

Por último debe hacer click en el botón registrar. El sistema validará los datos ingresados y le informará en caso de alguna inconsistencia.
